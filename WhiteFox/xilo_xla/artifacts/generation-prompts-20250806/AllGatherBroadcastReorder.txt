The model should contain the following pattern:

```
t1 = tf.broadcast_to(input_tensor, ...)
t2 = tf.all_gather(t1, ...)
```

The pattern describes that there is an `all_gather` operation in the model, which takes the output of a `broadcast` operation as its operand. The `all_gather` operation should be an array, and the `broadcast` operation should be the first operand of the `all_gather` operation.

The optimization pass `AllGatherBroadcastReorder` is triggered when the model contains an `all_gather` operation that is fed by a `broadcast` operation. The optimization pass will reorder these operations if there are any remaining uniform dimensions and their size is greater than 1. 

The optimization pass will transform the `all_gather(broadcast(x))` to `broadcast(all_gather(x))`. There are two cases here:

1. If the `all_gather` happens along one of the non-uniform dimensions of the `broadcast`, then issue `all_gather(x)` and then a `broadcast`.

2. If the `all_gather` happens along one of the uniform dimensions of the `broadcast`, that dimension does not exist in `x`. In this case, a reshape operation is added to add a leading '1' dimension, then the `all_gather` operation is performed, followed by a `broadcast` operation which matches the existing `all_gather` shape, except the `all_gather` dim is split into `[ag_factor, ag_dim_size/ag_factor]`. Finally, a reshape operation is performed to "flatten" the `[ag_factor, ag_dim_size/ag_factor]` to just `ag_dim_size`.
