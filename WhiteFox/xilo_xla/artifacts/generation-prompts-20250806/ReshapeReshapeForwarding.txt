The model should contain the following pattern:
```
t1 = tf.reshape(input_tensor, ...)
t2 = tf.reshape(t1, input_tensor.shape)
```
The pattern describes that there are two reshape operators in the model. The first `reshape` operator transforms a tensor input `input_tensor` from `input_tensor.shape` to any new shape, and the second `reshape` operator transforms the output of first `reshape` back to `input_tensor.shape`. 

In other words, the optimization pass `ReshapeReshapeForwarding` is triggered when a tensor is reshaped twice consecutively, with the second reshape operation reversing the first one, effectively making the two reshape operations redundant.
