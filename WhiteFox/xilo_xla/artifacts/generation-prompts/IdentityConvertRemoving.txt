### Please generate one valid TensorFlow model that satisfies requirements below.
You should only use public TensorFlow APIs. The model can be used as the input to trigger the optimization pass `IdentityConvertRemoving` in TensorFlow XLA.

# Description
The model should contain a type-cast operation where the input and output data types
are identical:
```
x = tf.constant([1.0, 2.0, 3.0], dtype=tf.float32)
y = tf.cast(x, dtype=tf.float32)   # identity cast: float32 -> float32
```
The optimization removes redundant `tf.cast` calls where the source and destination
data types are the same, directly forwarding the input tensor.
